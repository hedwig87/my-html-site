팀 빌려온 고양이

오승근 최민하 강지혁

LG webOS 딩벳을 쉽고 빠르게 검색하고 생성할 수 있는 AI 기반 서비스입니다. 사용자가 Flutter로 구현된 인터랙티브 캔버스에 그린 스케치를 CLIP 모델로 가장 유사한 딩벳을 검색하고, 없을 경우 LG 스타일에 맞춘 새로운 딩벳을 생성합니다. 벡터는 사전 계산 및 저장되어 빠른 검색을 지원합니다. CustomPainter와 RepaintBoundary를 사용해 드로잉을 부드럽게 처리하고 서버로 전송하며, Riverpod와 클린 아키텍처 기반으로 구성해 유지보수성과 확장성을 높였습니다. 백엔드는 FastAPI 기반으로 GCP Cloud Run 서버리스 환경에 배포돼 자동 확장과 Secret Manager를 통한 보안 관리가 가능하게 설계했습니다.

팀 C-4Bomb

김동영 이연주 정경환 조재용

직접 설계한 멀티프로세싱 운영체제(OS)를 이용하여 벤치마크 애플리케이션을 통해 CPU 성능을 평가하고 GUI 그래프로 나타내어 확인할 수 있는 프로젝트입니다. 객관적인 성능 확인을 위해 오픈소스 임베디드 시스템 벤치마크인 EmBench를 타겟 보드에 이식하였으며 그 결과를 실시간 그래프로 확인할 수 있습니다. 자체 구현한 OS에서는 Context switching, Demand paging, System call을 구현하여 다수의 애플리케이션을 동시에 실행할 수 있는 멀티태스킹 환경을 제공합니다. 이러한 기술을 바탕으로 필요한 성능을 충족하고 효율적인 임베디드 보드를 선택할 수 있도록 인사이트를 제시합니다.
